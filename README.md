LOS操作系统
==================

LOS是一个操作系统, 用于学习操作系统的基本理论. 操作系统同时具有两个身份: 一方面, 操作系统作为硬件的抽象, 为上层的应用程序提供更容易使用的接口(例如直接操作磁盘和使用文件这种抽象概念), 另一方面, 操作系统作为资源管理者, 管理底层的硬件, 并为上层的多个用户/程序提供资源.
